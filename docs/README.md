# Gilga

> 登録なし。開くだけ。もう会話が始まってる。

## これは何？

**Gilga**（ジルガ）は、アカウント登録なしで使えるチャット＆SNSアプリ。

ダウンロードして開く。それだけ。
メールアドレスも電話番号もいらない。
開いた瞬間、もう誰かが喋ってる。

## なぜ作った？

LINEは友だち登録がいる。
Discordはサーバーを選ばないといけない。
Xはアカウントを作らないと見れない。

**面倒くさい。**

ただ喋りたいだけなのに、なんでこんなに手続きがいるんだ？

Gilgaは「通りすがりの会話」を取り戻す。
昔の掲示板、昔のチャット、昔のインターネット。
名乗らなくても参加できた、あの感じ。

## 特徴

### 登録ゼロ
メールアドレス不要。電話番号不要。本名不要。
開いたら使える。

### 常駐オーバーレイ
ゲーム中でも、作業中でも、画面の端にチャットが流れる。
邪魔にならない。でも、そこにいる。

### 全部見える
チャットも、投稿も、全部が一つの流れ。
「どこに行けばいいか分からない」がない。
開けば、全部ここにある。

## 思想

Gilgaは人間の欲望を肯定する。

- **目立ちたい？** いいぞ。発言しろ。
- **好きな人と集まりたい？** そのためにある。
- **宣伝したい？稼ぎたい？** 堂々とリンクを貼れ。恥じるな。
- **暇つぶししたい？** 開けば誰かいる。

投げ銭乞食みたいな機能はない。
稼ぎたいなら、自分のサイトに誘導しろ。
それが人間らしい商売だ。

## 名前の由来

**ギルガメッシュ叙事詩**から。

人類最古の文学。
不死を求め、友を愛し、欲望のままに生きた王の物語。

神に逆らった。
永遠を求めた。
友のために泣いた。
**人間らしく生きた。**

Gilgaは、人間讃歌のアプリだ。
