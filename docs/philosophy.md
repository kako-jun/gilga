# 設計思想

## ルネッサンス的人間讃歌

中世ヨーロッパは人間の欲望を罪とした。
現代のSNSも同じことをしている。

```
「コミュニティガイドライン」= 欲望の抑圧
「利用規約違反」= 現代の異端審問
「アカウント凍結」= 社会的処刑
```

Gilgaはルネッサンス。
人間の欲望を再び肯定する。

## 肯定する欲望

### 承認欲求
目立ちたい。認められたい。これは恥ずかしいことじゃない。
人間が社会的動物である証拠だ。

Gilgaでは発言が流れる。人に見られる。反応がもらえる。
それでいい。

### 帰属欲求
好きな人たちと集まりたい。同じ趣味の人と話したい。
群れたいのは本能だ。

Gilgaでは全員が同じ場所にいる。
見つけたい人は見つかる。

### 金銭欲
稼ぎたい。自分の商品を売りたい。サイトに人を呼びたい。
これも人間の正当な欲求だ。

**Gilgaに投げ銭機能はない。**
乞食のように小銭を集めるUIは入れない。

稼ぎたいなら、堂々と自分のリンクを貼れ。
「こういうサービスやってます」「買ってください」
それが人間らしい商売だ。隠すな。

### 暇つぶし
生産性なんかなくていい。
ただ時間を潰したい。誰かの声を聞いていたい。
それも立派な欲求だ。

Gilgaを開けば、誰かがいる。
何もしなくても、会話が流れていく。

## 否定するもの

### 偽善
「分散だから正義」「検閲耐性が大事」
技術者の自己満足。一般人には関係ない。

ユーザーは便利だから使う。
思想なんか押し付けるな。

### 乞食根性
「投げ銭お願いします」「サポートしてください」
大企業に反発しておいて、金は欲しいのか。

稼ぎたいなら堂々と商売しろ。
投げ銭ボタンで小銭を集めるのは情けない。

### 過度な分散
「誰もHubになりたくない」という態度。
結果、人が分散して過疎に見える。
新規が来ない。広まらない。負け組の思考。

Gilgaは統合する。
全部を一つの流れにまとめる。
「ここに来れば全部見える」を作る。

## 技術は手段

ユーザーが知る必要のないこと：
- プロトコルの名前
- 暗号技術の仕組み
- 分散の仕組み

ユーザーが知ればいいこと：
- 登録がいらない
- 開けば使える
- 人がいる

HTTPSを意識してウェブを使う人はいない。
Gilgaも同じ。裏で何が動いていようと関係ない。

## ターゲット

**LINEが使えるレベルのリテラシーの人**

技術オタクじゃない。
「プロトコル」なんて言葉を知らない。
ただ、友達と喋りたい。暇を潰したい。

そういう人が自然に使ってる状態を目指す。

Z世代が「なんか使ってる」と言い始めたら勝ち。

## 名前に込めた意味

ギルガメッシュ。
人類最古の英雄。

- 神に逆らった
- 不死を求めた
- 友エンキドゥを愛した
- その死を嘆いた
- 最後は人間として生きることを受け入れた

欲望のままに生きて、それでも最後は人間だった。
Gilgaはその精神を継ぐ。
